# Single Sales Factor Apportionment (SSFA) Project

This repository contains replication materials for  
**“The Impact of Single Sales Factor Apportionment on State Tax Revenues: Short- and Long-Run Effects”** (2025).
[SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=5357629)

## Structure

- `analysis/` – Organized by paper sections (e.g., `2.4_fixed_factors_logit`, `7.3_synthetic_did`).
- `data/raw/` – Original state-level datasets.
- `data/filtered/` – Cleaned datasets prepared for analysis.
- `scripts/` – General scripts not yet tied to a specific section.
- `archive/` – Miscellaneous or historical files (not used in replication).
- `output/` – Tables and figures saved by scripts.
